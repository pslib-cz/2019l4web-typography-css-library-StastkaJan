# Typography.css
## CSS typography template for your blog

### Included HTML elements
- p
- blockquote
- big
- cite
- em
- small
- a
- h1, h2, h3, h4, h5, h6
- ul, ol
- code
- pre
- img
- button

### Instalation
1. Download typography.css
2. Link it to your page
3. Complete - all included HTML elements are now edited

### Usage
yYu don't have to do anything.
